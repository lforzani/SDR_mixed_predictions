
This folder contains the MATLAB source code for our submission "Sparse Ising Model with Covariates".
The codes that start with "main_" runs all the simulation study under various settings. Data analysis is contained in "data_DNA_select.m". 
The code that fits the joint and separate approach are "Solpath.m" and "Solpath_idv.m".
Most of the major codes are moderately commented.
