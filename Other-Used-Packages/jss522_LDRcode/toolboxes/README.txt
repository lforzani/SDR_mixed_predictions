- DESCRIPTION
--------------------------------------------------------------------------------
This folder contains third-party routines that are used in the LDR package with
permission from the authors.

# SG_MIN: a toolkit by Lippert that provides a Matlab implementation of 
optimization algorithms over the Grassmann and Stieffel manifolds.

For more details, see Lippert, R. and Edelman, A. (2000). Nonlinear eigenvalue 
problems with orthogonality constraints. In Bai, Z., Demmel, J., Dongarra, J., Ruhe, 
A. and van der Vorst, H: Templates for the Solution of Algebraic Eigenvalue Problems: 
A Practical Guide. Philadelphia, SIAM.


# LIGHTSPEED: a toolkit by TOM MINKA that provides efficient implementations of 
            common functions to deal with matrices.

For more information, visit: 
http://research.microsoft.com/en-us/um/people/minka/software/lightspeed/
